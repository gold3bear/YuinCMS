解釋一下你當前所使用的模板系統是怎麼回事

模板系統html結構其實很簡單，你之所以會看到2張圖片同置一行，是因為一個叫row的div包住2個叫grid6的div（圖片就在裡面，每個grid6包住1個圖片），為什麼是數字6呢？因為row默認是12，那麼只有讓2個grid加起等於12才能滿足要求嘛，當然也可以是3個grid4，或則6個grid2。這個看需求了。如果你想圖片居中，那也很好辦，只需要吧grid12包在row裡面，給grid12 後面加上center即可。
